# 🚗 Car Delivery Game (Unity)

A simple **2D Car Delivery Game** built using **Unity and C#**.
The player controls a car to pick up items and deliver them while avoiding obstacles.

---

## 🎮 Gameplay

* Drive the car using keyboard controls
* Pick up items (banana 🍌)
* Deliver them to customers 👤
* Gain speed boost ⚡ and manage movement

---

## ✨ Features

* 🚘 Smooth car movement with rotation
* ⚡ Speed boost system on collision
* 📦 Pickup & delivery mechanic
* 💥 Collision detection with environment
* 🎯 Simple and interactive gameplay

---

## 🎹 Controls

| Key | Action        |
| --- | ------------- |
| W   | Move Forward  |
| S   | Move Backward |
| A   | Turn Left     |
| D   | Turn Right    |
